# Бычатник 2 — учебный проект на Python

Веб-приложение для учета быков по секциям:
- хранение карточек быков;
- история веса;
- импорт перевески из Excel;
- массовый перевод между секциями;
- выбытие с маркером причины;
- записи о здоровье и лечении;
- отчеты, рейтинг слабых/лучших по привесу, расчет быкодней;
- выгрузка отчетов в Excel и PDF.

## Технологии

- Python
- Django
- SQLite (на старте)
- pandas + openpyxl (Excel)
- reportlab (PDF)

## Быстрый старт

1. Создать окружение и установить зависимости:

   ```bash
   python3 -m venv .venv
   .venv/bin/pip install -r requirements.txt
   ```

2. Применить миграции:

   ```bash
   .venv/bin/python manage.py migrate
   ```

3. Создать суперпользователя (для админки):

   ```bash
   .venv/bin/python manage.py createsuperuser
   ```

4. Запустить сервер:

   ```bash
   .venv/bin/python manage.py runserver
   ```

5. Открыть:
   - приложение: `http://127.0.0.1:8000/`
   - админка: `http://127.0.0.1:8000/admin/`

## Формат Excel для перевески

- Колонка A: `бивайка` (ровно 8 символов)
- Колонка B: `номер быка` (ровно 6 символов)
- Колонки C, D, E...: даты перевески в формате `ДД.ММ.ГГГГ`
- В ячейках под датами: вес (целые кг)

Если номер не найден, запись попадает в список проблем импорта с подбором похожих номеров.

# Учет быков (учебный проект)

Проект на Django для учета быков по секциям: поступления, перемещения, перевески и выбытия.

## Что реализовано

- 8 секций площадки (`L1-L4`, `R1-R4`), команда инициализации `init_sections`.
- Карточка быка с уникальным номером (`external_id`), где:
  - первые 8 символов - "бивайка";
  - последние 6 символов - "номер быка".
- История взвешиваний (Excel + ручной ввод).
- Перемещение между секциями в любую дату.
- Выбытие с маркером причины для статистики:
  - планово на мясокомбинат;
  - срочно на убой;
  - другое.
- Импорт Excel с обработкой опечаток в номерах:
  - если номер не найден, создается проблемная строка;
  - система предлагает похожие номера;
  - админ вручную сопоставляет запись.
- Главная сводка и рейтинги:
  - слабый/лучший набор по абсолютному привесу;
  - слабый/лучший набор по среднесуточному привесу.

## Формат Excel

- Колонка A: полный номер быка (например `by000160474233`).
- Колонка B: вес на предыдущую дату.
- Колонка C: вес на текущую дату.
- Даты взвешивания задаются в форме импорта.

## Быстрый старт

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python manage.py migrate
.venv/bin/python manage.py init_sections
.venv/bin/python manage.py createsuperuser
.venv/bin/python manage.py runserver
```

Открой:
- `http://127.0.0.1:8000/` - сайт;
- `http://127.0.0.1:8000/admin/` - админка.

## Важная заметка
